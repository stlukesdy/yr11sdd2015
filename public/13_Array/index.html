13 Array

A simple array is a grouped list of similar variables, all of which are used for the same purpose and are of the same data type. Individual elements are accessed using an index, which is a simple integer variable.

Before using an array in a program, some languages first require that it must be dimensioned (that is, defined) in order to allocate sufficient memory for the specified number of elements. A statement such as the following is a typical example of the required code.

DIM Names (20) as string

Problem

Use an array to store multiple names. Print the third name. Change the second name, and print all names.

Pseudocode

nameArray(1) = "John"
nameArray(2) = "Hassan"
nameArray(3) = "Chris"
nameArray(4) = "Mary"

Display "The third name is: "; nameArray(3)

nameArray(2) = "Adam"

FOR i = 1 to 4 STEP 1
  Display "nameArray at index", i," = ", nameArray(i)
NEXT i

Javascript

var nameArray = [];
//Note: javascript arrays start at index 0

nameArray[0] = "John";
nameArray[1] = "Hassan";
nameArray[2] = "Chris";
nameArray[3] = "Mary";

console.log("The third name is: " + nameArray[2]);

nameArray[1] = "Adam";

for (var i = 0; i < 4; i = i + 1) {
  console.log("nameArray at index" + i + " = " + nameArray[i]);
}
