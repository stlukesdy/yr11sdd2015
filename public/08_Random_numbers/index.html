08 Random numbers

Problem

Print a random number between 1 and 10.

Pseudocode

The following generic statement is used to generate a random number between two values (HighValue and LowValue):

R = Random (HighValue – LowValue) + LowValue


r = Random(9) + 1
Display r

Javascript

Math.random(); returns a random number between 0 (inclusive) and 1 (exclusive). Math.floor(1.6); rounds a number downward to its nearest integer. 

var r = Math.floor((Math.random() * 10) + 1);
console.log(r);
